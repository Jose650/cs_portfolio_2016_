Hardware

The motherboard is the main circut board. It is used for other mirco computers as well.

Central Processing Unit (CPU)

A CPU stands for Computer Processing Unit which helps the computer process everything

RAM (Random Access Memory)

A RAM stands for Random Access Memory it is basically the memory of the whole computer system

Hard Drive

A Hard Drive is the storage device containing a read writing program.

BUS

A set of conductors carrying data and control signals in a computer system, to which pieces of equipment may be connected in parallel.

KeyBoard

A panel of keys that operate a computer.

Mouse

Using a mouse to move a cursor on a computer screen.

Monitor

A screen that displays an image generated by a computer.
