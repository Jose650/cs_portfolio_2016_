How to make an drop down image

To make a Image drop down you'll need to find a image that you would like to use to make it drop down text when you click on it. I decided to make this because I felt it wasn't too diffcult or easy to do. For example, I will use a picture of Deadshot from the Justice League Tv Show. 

                 
<img id="Deadshotpic" src="http://vignette1.wikia.nocookie.net/warner-bros-characters/images/4/46/Deadshot.png/revision/latest?cb=20140821184138"
         width="300px" height="250px">
         <div style="display:none" id="Deadshotinfo">

Now you'll have to add Id, div style tag so it can display the text inside the image source tag. Make sure that the div style tag closes at the end of the Paragraph tag. In the Id part of the tag make sure you name the image with pic next to it with quotations in the beginning and end. For the div style tag makes sure you put display:none with quotations and out the id tag again at the end like with the first one but make sure u put info instead of pic because it you want to show the paragraph you wrote about the image you are showing and close it with out the source image code. 

                  
 var Deadshotpic = document.getElementById("Deadshotpic");
var Deadshotinfo = document.getElementById("Deadshotinfo");
  
  
  Deadshotpic.addEventListener('click',function(){
 
        
        
        if(Deadshotinfo.style.display ==="none"){
        Deadshotinfo.style.display = "block";
    }
    else {
        Deadshotinfo.style.display = "none";
    }
});

You need to put var and getElementById tag inside the Javascript file. Which you'll have to link a javasrcipt file to the image which has html file in already.

You'll need to make two of the var and getElementById tags one for your images name with pic in it and info in it was well so it can read it from the html file.

Next you'll have to make a click function so when you hover over the image and click it it will be clickable. it need to have addEventListener so it know when your clicking.

For the final part you'll need to create a if else statement so it can display the paragraph with the div style tags
