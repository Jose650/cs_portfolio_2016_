I decided to create an Todo list for my How To Page because I thought it was be nice to make a list of things I need to do in order to be organized. The Todo list is a list of tasks that are needed to be completed that are organized in particular order of priority.

To create the Todo List you'll need to use HTML and Javascript. You use the HTML in order to create the actual code of the Todo List itself and the Javascript is used to program the buttons what to do itself.

The input id's creates a text input box on the screen displaying the text provided. You need to make box for the things you are listing which is the "textinput" this will display inside the first box. the code type="text" will let the html know its a text.

The next step is to create the buttons for the inputs for the submit and reset. The Submit button will run the textinput so it can create the first listed item part of the todo list. The code to make the button is id="buttoninput" type="button" value="Submit">

The thrid step is to make the another button for the reset. The reason for making the reset button is if clear off the mistakes someone could make while creating the todo list. Now in we will be using the same input id code as the Submit button used but instead call it id="button2input" because this is the 2nd button input but the value would be differnet since its not a submit button its a reset so the code format would be type="button" value="Reset"

Now the final step for our buttons is the select button which changes the colors of the todo list text. The code for the select input is
